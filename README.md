# Repartir

## Ambiente

- JDK 17
- Docker
- Chrome
- Node 18

## Build & Run

### Build del proyecto

```
./gradlew build
```

### Para levantar el ambiente local

```
./gradlew bootRun
```

### Para levantar el frontend angular

```
./iniciar-frontend
```

## Acceso web

http://localhost:8080/

## Open API

http://localhost:8080/swagger-ui.html
