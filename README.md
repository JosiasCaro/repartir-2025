# Repartir

## Ambiente

 * JDK 17
 * Docker
 * Chrome

## Ejecutar pruebas

Para ejecutar todas las pruebas

```
./gradlew check --info
```

## Para levantar en el ambiente local

```
./gradlew bootRun
```

## Para levantar el frontend angular

```
./iniciar-frontend
```

## Para ejecutar pruebas ui-angular

```
./tests js unit
```

## Acceso web

http://localhost:8080/

## Open API

http://localhost:8080/swagger-ui.html
